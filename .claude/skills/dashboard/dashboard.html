First-time setup

The dashboard reads everything from your local persistence/ directory — account, positions, configs, snapshots, pool. One grant is needed.

Pick the ClaudeTrading repo root

Pick the folder that contains persistence/, lib/, and CLAUDE.md — i.e. the top-level ClaudeTrading directory, NOT a subfolder like .claude/skills/dashboard/. We verify the pick before saving it.

Grants the page read+write access for configs and snapshots. FileSystemDirectoryHandle stored in IndexedDB; never uploaded. Browsers block file://-to-file:// reads as a security feature, so this explicit grant is the sandbox-friendly path.

Optional, later: if you want live news in the Market intel tab, click 🔑 Creds in the header any time to paste your Alpaca paper key. The dashboard does not require credentials to function — tabs 1–4 work fully without them, and the Congressional-trades section in tab 5 is fetched without auth.

Alpaca credentials (optional)

Stored in localStorage. Sent only to Alpaca's API. Used by the news fetch in the Market intel tab. Leave blank and click Save to disable news fetching.
